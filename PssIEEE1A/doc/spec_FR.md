Description globale : **Adapté des modèles de données CIM. La classe représente le modèle de stabilisateur de système électrique PSS1A de type IEEE Std 421.5-2005. PSS1A est la forme généralisée d'un PSS avec une seule entrée. Certains signaux d'entrée courants du stabilisateur sont la vitesse, la fréquence et la puissance.  Référence : IEEE 1A 421.5-2005 Section 8.1.**
