Descrizione globale: **Adattato dai modelli di dati CIM. La classe rappresenta il modello di stabilizzatore del sistema di alimentazione PSS1A di tipo IEEE Std 421.5-2005. PSS1A è la forma generalizzata di un PSS con un singolo ingresso. Alcuni segnali di ingresso comuni dello stabilizzatore sono la velocità, la frequenza e la potenza.  Riferimento: IEEE 1A 421.5-2005 sezione 8.1.**  
versione: 0.0.1
