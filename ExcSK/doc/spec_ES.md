Descripción global: **Adaptado de los modelos de datos CIM. Modelo del sistema de excitación eslovaco.  En este modelo se incluyen el limitador de subexcitación y el control de la tensión secundaria. Cuando se utiliza este modelo, no puede haber un modelo separado de limitador de subexcitación o de controlador de VAr.**
