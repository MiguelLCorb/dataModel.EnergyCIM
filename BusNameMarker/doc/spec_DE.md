Globale Beschreibung: **Angepasst von CIM-Datenmodellen. Wird verwendet, um Benutzer-Standardnamen auf Topologie-Busse anzuwenden. Wird in der Regel für die Generierung von Bus-/Abzweigfällen verwendet. Assoziiert mit einer oder mehreren Klemmen, die normalerweise mit dem Busnamen verbunden sind. Die assoziierten Klemmen sind normalerweise durch nicht gehaltene Schalter verbunden. Bei einer Ringsammelschienenkonfiguration sind normalerweise alle Sammelschienenklemmen im Ring verbunden. Bei einem Trennschalter und einem Halbschema werden normalerweise beide Sammelschienen verbunden.  Bei einer Ringsammelschiene sind in der Regel alle Sammelschienen miteinander verbunden.  Bei einer "geraden" Sammelschienenkonfiguration wird normalerweise nur die Hauptklemme an der Sammelschiene zugeordnet.**  

## Liste der Eigenschaften  

- `ReportingGroup`: Die Busnamenmarker, die zu dieser Meldegruppe gehören. Voreinstellung: Keine  - `Terminal`: Die Klemmen, die mit diesem Busnamenmarker verbunden sind. Voreinstellung: 'list'  - `address`: Die Postanschrift  - `alternateName`: Ein alternativer Name für diesen Artikel  - `areaServed`: Das geografische Gebiet, in dem eine Dienstleistung oder ein angebotener Artikel erbracht wird  - `dataProvider`: Eine Folge von Zeichen zur Identifizierung des Anbieters der harmonisierten Dateneinheit.  - `dateCreated`: Zeitstempel der Entitätserstellung. Dieser wird in der Regel von der Speicherplattform zugewiesen.  - `dateModified`: Zeitstempel der letzten Änderung der Entität. Dieser wird in der Regel von der Speicherplattform vergeben.  - `description`: Eine Beschreibung dieses Artikels  - `id`: Eindeutiger Bezeichner der Entität  - `location`: Geojson-Referenz auf das Element. Es kann Punkt, LineString, Polygon, MultiPoint, MultiLineString oder MultiPolygon sein  - `name`: Der Name dieses Artikels.  - `owner`: Eine Liste mit einer JSON-kodierten Zeichenfolge, die auf die eindeutigen Kennungen der Eigentümer verweist  - `priority`: Priorität der Busnamenmarkierung für die Verwendung als Topologiebusname.  Benutzen Sie 0 für "egal".  Verwenden Sie 1 für die höchste Priorität.  Verwenden Sie 2, wenn die Priorität kleiner als 1 ist, und so weiter. Voreinstellung: 0  - `seeAlso`: Liste von URLs, die auf zusätzliche Ressourcen zu dem Artikel verweisen  - `source`: Eine Folge von Zeichen, die die ursprüngliche Quelle der Entitätsdaten als URL angibt. Empfohlen wird der voll qualifizierte Domänenname des Quellanbieters oder die URL des Quellobjekts.  - `type`: NGSI-Typ. Es muss BusNameMarker sein
