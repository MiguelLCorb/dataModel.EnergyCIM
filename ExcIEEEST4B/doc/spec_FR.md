Description globale : **Adapté des modèles de données CIM. La classe représente le modèle IEEE Std 421.5-2005 de type ST4B. Ce modèle est une variante du modèle de type ST3A, avec un bloc régulateur proportionnel plus intégral (PI) remplaçant la caractéristique de régulateur à retardement qui se trouve dans le modèle ST3A. Les systèmes d'excitation à source potentielle et à source composée sont modélisés.  Les blocs régulateurs PI ont des limites de non-récupération qui sont représentées. Le régulateur de tension de ce modèle est typiquement implémenté numériquement.  Référence : IEEE Standard 421.5-2005 Section 7.4.**  
version : 0.0.1

Adapté des modèles de données CIM et de CIMpy - [https://github.com/sogno-platform/cimpy](https://github.com/sogno-platform/cimpy). Les classes python sur lesquelles ce modèle est basé ont été développées par les entités suivantes : Institute for Automation of Complex Power Systems (ACS), EON Energy Research Center (EONERC) et RWTH University Aachen, Allemagne.
