Descripción global: **Adaptado de los modelos de datos CIM. Modelo detallado del sistema de excitación - ELIN (VATECH).  Este modelo representa un sistema de excitación totalmente estático. Un controlador PI de tensión establece una consigna de corriente de campo deseada para un controlador proporcional de corriente. El integrador del controlador PI tiene una entrada de seguimiento para ajustar su señal a la corriente de campo actual.  Modelos de estabilizador del sistema de potencia utilizados junto con este modelo de sistema de excitación: PssELIN2, PssIEEE2B, Pss2B.**  
versión: 0.0.1
