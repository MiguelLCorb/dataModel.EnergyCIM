Entité : GovGAST3  
=================  
[Licence ouverte](https://github.com/smart-data-models//dataModel.EnergyCIM/blob/master/GovGAST3/LICENSE.md)  
Description globale : **Adapté des modèles de données CIM. Turbogaz générique avec contrôleur d'accélération et de température.**  

## Liste des biens  

- `address`: L'adresse postale.  - `alternateName`: Un autre nom pour cet article  - `areaServed`: La zone géographique où un service ou un article offert est fourni  - `bca`: Consigne de limite d'accélération (Bca).  Unité = 1/s.  Valeur typique = 0,01. Valeur par défaut : 0,0  - `bp`: Droop (bp).  Valeur typique = 0,05. Valeur par défaut : 0,0  - `dataProvider`: Une séquence de caractères identifiant le fournisseur de l'entité de données harmonisées.  - `dateCreated`: Horodatage de la création de l'entité. Il est généralement attribué par la plate-forme de stockage.  - `dateModified`: Horodatage de la dernière modification de l'entité. Il est généralement attribué par la plate-forme de stockage.  - `description`: Une description de cet article  - `dtc`: Variation de la température des gaz d'échappement due à l'augmentation du débit de carburant de 0 à 1 PU (deltaTc).  Valeur typique = 390. Valeur par défaut : 0,0  - `id`: Identifiant unique de l'entité  - `ka`: Débit minimum de carburant (Ka).  Valeur typique = 0,23. Valeur par défaut : 0,0  - `kac`: Retour d'information sur le système de carburant (K).  Valeur typique = 0, par défaut : 0,0  - `kca`: Gain intégral de contrôle de l'accélération (Kca). Unité = 1/s.  Valeur typique = 100. Valeur par défaut : 0,0  - `ksi`: Gain du bouclier anti-radiation (Ksi).  Valeur typique = 0,8. Valeur par défaut : 0,0  - `ky`: Coefficient de la fonction de transfert du positionneur de la vanne de carburant (Ky).  Valeur typique = 1. Valeur par défaut : 0,0  - `location`:   - `mnef`: Valeur d'erreur négative maximale du débit de carburant (MN).  Valeur typique = -0,05. Valeur par défaut : 0,0  - `mxef`: Valeur d'erreur positive maximale du débit de carburant (MX).  Valeur typique = 0,05. Valeur par défaut : 0,0  - `name`: Le nom de cet article.  - `owner`: Une liste contenant une séquence de caractères codés en JSON faisant référence aux Ids uniques du ou des propriétaires  - `rcmn`: Débit minimum de carburant (RCMN).  Valeur typique = -0,1. Valeur par défaut : 0,0  - `rcmx`: Débit maximal de carburant (RCMX).  Valeur typique = 1. Valeur par défaut : 0,0  - `seeAlso`: liste d'uri pointant vers des ressources supplémentaires sur le sujet  - `source`: Une séquence de caractères donnant comme URL la source originale des données de l'entité. Il est recommandé d'utiliser le nom de domaine complet du fournisseur de la source, ou l'URL de l'objet source.  - `tac`: Constante de temps de contrôle du carburant (Tac).  Valeur typique = 0,1. Valeur par défaut : 0  - `tc`: Constante de temps du volume de décharge du compresseur (Tc).  Valeur typique = 0,2. Valeur par défaut : 0  - `td`: Gain dérivé du régulateur de température (Td).  Valeur typique = 3,3. Valeur par défaut : 0  - `tfen`: La température nominale d'échappement de la turbine correspond à Pm=1 PU (Tfen).  Valeur typique = 540. Valeur par défaut : 0,0  - `tg`: Constante de temps du régulateur de vitesse (Tg).  Valeur typique = 0,05. Valeur par défaut : 0  - `tsi`: Constante de temps du bouclier anti-radiation (Tsi).  Valeur typique = 15. Valeur par défaut : 0  - `tt`: Taux d'intégration du régulateur de température (Tt).  Valeur typique = 250. Valeur par défaut : 0,0  - `ttc`: Constante de temps du thermocouple (Ttc).  Valeur typique = 2,5. Valeur par défaut : 0  - `ty`: Constante de temps du positionneur de vanne de carburant (Ty).  Valeur typique = 0,2. Valeur par défaut : 0  - `type`: Type NGSI. Il doit être de type GovGAST3    
Propriétés requises  
Ce modèle de données est une conversion directe du modèle commun d'information (CIM) spécifié par la norme IEC61970 en modèles de données intelligents. Les classes python sur lesquelles ce modèle est basé ont été développées par ces entités : Institute for Automation of Complex Power Systems (ACS), EON Energy Research Center (EONERC) et RWTH University Aachen, Allemagne. Certaines propriétés peuvent avoir un mauvais type. C'est le cas, pelase soulever un problème ou envoyer un mail à alberto.abella@fiware.org
