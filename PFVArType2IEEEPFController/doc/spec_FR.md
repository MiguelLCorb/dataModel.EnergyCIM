Description globale : **Adaptée des modèles de données CIM. La classe représente le contrôleur IEEE PF de type 2 qui est un contrôleur de type point de sommation et constitue la boucle extérieure d'un système à deux boucles. Ce contrôleur est implémenté comme un contrôleur lent de type PI. Le régulateur de tension constitue la boucle intérieure et est implémenté comme un régulateur rapide.  Référence : IEEE Standard 421.5-2005 Section 11.4.**.  
version : 0.0.1
