Globale Beschreibung: **Abgeleitet aus CIM-Datenmodellen. Beschreibt die Übersetzung eines Wertesatzes in einen Namen und dient der Erleichterung von Cusom-Übersetzungen. Jedes ValueAliasSet hat einen Namen, eine Beschreibung usw. Ein bestimmtes Measurement kann einen diskreten Zustand wie Open, Closed, Intermediate etc. repräsentieren. Dies erfordert eine Übersetzung von der MeasurementValue.value-Nummer in einen String, z. B. 0->'Invalid', 1->'Open', 2->'Closed', 3->'Intermediate'. Jedes ValueToAlias-Mitglied in ValueAliasSet.Value beschreibt eine Zuordnung für einen bestimmten Wert zu einem Namen.
