<!-- 20-Description -->  
글로벌 설명: **CIM 데이터 모델에서 수정됨. 전압 또는 유량과 같은 전력 시스템 양을 제어하기 위해 함께 작동하는 장비 집합을 지정합니다.  제어 장비에서 멀리 떨어진 곳에 위치한 제어 단자를 지정하여 원격 버스 전압 제어가 가능합니다. 서로 다른 유형의 여러 장비가 동일한 단자를 제어하는 경우 해당 단자에는 하나의 RegulatingControl만 존재해야 합니다. 이러한 장비가 제어에 참여하는 경우 가장 구체적인 하위 유형(예: 탭 체인저의 경우 TapChangerControl)을 사용해야 합니다. 흐름 제어의 경우 부하 부호 규칙이 사용됩니다. 즉, 양수 부호는 TopologicalNode(버스)에서 전도성 장비로의 흐름을 의미합니다.**  
버전: 0.0.1
