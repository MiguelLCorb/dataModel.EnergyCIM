Descrizione globale: **Adattato dai modelli di dati CIM. La classe rappresenta il modello IEEE Std 421.5-2005 tipo DC4B. Questi sistemi di eccitazione utilizzano un eccitatore a commutatore in c.c. controllato dal campo con un regolatore di tensione ad azione continua con alimentazioni ottenute dal generatore o dal bus ausiliario.  Riferimento: Standard IEEE 421.5-2005 Sezione 5.4.**
