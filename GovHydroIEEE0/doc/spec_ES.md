Descripción global: **Adaptado de los modelos de datos CIM. Modelo simplificado de turbina-gobernador IEEE.  Se utiliza para reguladores de turbina mecánicos-hidráulicos y electro-hidráulicos, con o sin retroalimentación de vapor. Los valores típicos indicados son para el modelo mecánico-hidráulico.  Ref**  

## Lista de propiedades

Por defecto: 0,0  - `location`: Referencia Geojson al elemento. Puede ser Point, LineString, Polygon, MultiPoint, MultiLineString o MultiPolygon  - `mwbase`: Base para los valores de potencia (MWbase) (> 0).  Unidad = MW. Por defecto: 0  - `t2`: Constante de tiempo del gobernador (T2.  Valor típico = 0. Por defecto: 0  - `t3`: Constante de tiempo del actuador de la puerta (T3).  Valor típico = 0,1. Por defecto: 0  - `t4`: Tiempo de arranque del agua (T4). Por defecto: 0  - `type`: Tipo de NGSI. Tiene que ser GovHydroIEEE0    
Propiedades requeridas  
Adaptado de los modelos de datos CIM y CIMpy - [https://github.com/sogno-platform/cimpy](https://github.com/sogno-platform/cimpy). Este modelo de datos es una conversión directa del Modelo de Información Común (CIM) especificado por la norma IEC61970 en modelos de datos inteligentes. Las clases de python en las que se basa este modelo fueron desarrolladas por estas entidades Institute for Automation of Complex Power Systems (ACS), EON Energy Research Center (EONERC) y RWTH University Aachen, Alemania. Algunas propiedades pueden tener un tipo incorrecto. Este es el caso, por favor, plantee una cuestión o envíe un correo a info@smartdatamodels.org.
