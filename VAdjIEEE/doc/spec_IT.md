Descrizione globale: **Adattato dai modelli di dati CIM. La classe rappresenta il regolatore di tensione IEEE che è usato per rappresentare il regolatore di tensione in un sistema di controllo del fattore di potenza o del var.  Riferimento: Standard IEEE 421.5-2005 Sezione 11.1.**  

## Elenco delle proprietà  

- `address`: L'indirizzo postale  - `adjslew`: Tasso al quale l'uscita del regolatore cambia ().  Unità = sec./PU.  Valore tipico = 300. Può essere Point, LineString, Polygon, MultiPoint, MultiLineString o MultiPolygon  - `name`: Il nome di questo articolo.  - `owner`: Una lista contenente una sequenza di caratteri codificata in JSON che si riferisce agli ID unici dei proprietari  - `seeAlso`: elenco di uri che puntano a risorse aggiuntive sull'elemento  - `source`: Una sequenza di caratteri che dà la fonte originale dei dati dell'entità come URL. Deve essere VAdjIEEE  - `vadjf`: Impostare in alto per fornire un aumento o un abbassamento continuo (). Predefinito: 0.0  - `vadjmax`: Uscita massima del regolatore ().  Valore tipico = 1.1. Predefinito: 0.0  - `vadjmin`: Uscita minima del regolatore ().  Valore tipico = 0.9. Predefinito: 0.0
