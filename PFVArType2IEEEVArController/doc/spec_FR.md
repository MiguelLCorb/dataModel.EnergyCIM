Description globale : **Adaptée des modèles de données CIM. La classe représente le contrôleur IEEE VAR de type 2 qui est un contrôleur de type point de sommation. Il constitue la boucle extérieure d'un système à deux boucles. Ce contrôleur est implémenté comme un contrôleur lent de type PI, et le régulateur de tension forme la boucle intérieure et est implémenté comme un contrôleur rapide.  Référence : IEEE Standard 421.5-2005 Section 11.5.**.
