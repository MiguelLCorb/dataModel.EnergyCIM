Globale Beschreibung: **Abgeleitet von CIM-Datenmodellen. Die Klasse repräsentiert das Modell IEEE Std 421.5-2005 Typ ST6B. Dieses Modell besteht aus einem PI-Spannungsregler mit einem Feldspannungsregler im inneren Regelkreis und einer Vorsteuerung. Der Feldspannungsregler implementiert eine proportionale Steuerung. Die Vorsteuerung und die Verzögerung im Rückführkreis erhöhen die Dynamik.  Referenz: IEEE-Norm 421.5-2005 Abschnitt 7.6.**
