Entität: ExcIEEEST6B  
====================  
[Offene Lizenz](https://github.com/smart-data-models//dataModel.EnergyCIM/blob/master/ExcIEEEST6B/LICENSE.md)  
Globale Beschreibung: **Abgeleitet von CIM-Datenmodellen. Die Klasse repräsentiert das Modell IEEE Std 421.5-2005 Typ ST6B. Dieses Modell besteht aus einem PI-Spannungsregler mit einem Feldspannungsregler im inneren Regelkreis und einer Vorsteuerung. Der Feldspannungsregler implementiert eine proportionale Regelung. Die Vorsteuerung und die Verzögerung im Rückführkreis erhöhen das dynamische Verhalten.  Referenz: IEEE Standard 421.5-2005 Abschnitt 7.6.**

Erforderliche Eigenschaften  
Dieses Datenmodell ist eine direkte Umsetzung des Common Information Model (CIM), das durch die Norm IEC61970 spezifiziert ist, in Smart Data Models. Die Python-Klassen, auf denen dieses Modell basiert, wurden von diesen Einrichtungen entwickelt Institut für Automatisierung komplexer Stromversorgungssysteme (ACS), EON Energy Research Center (EONERC) und RWTH Aachen, Deutschland. einige Eigenschaften können einen falschen Typ haben.
