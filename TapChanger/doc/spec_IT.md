Descrizione globale: **Adattato dai modelli di dati CIM. Meccanismo per cambiare la posizione del rubinetto dell'avvolgimento del trasformatore.  

## Elenco delle proprietà  

- `SvTapStep`: Lo stato del passo di commutazione associato al commutatore. Predefinito: Nessuno  - `TapChangerControl`: Il commutatore che partecipa a questo schema di controllo di regolazione. Predefinito: Nessuno  - `TapSchedules`: Un TapSchedule è associato a un TapChanger. Predefinito: 'elenco'.  - `address`: L'indirizzo postale  - `alternateName`: Un nome alternativo per questa voce  - `areaServed`: L'area geografica in cui viene fornito un servizio o un articolo offerto  - `controlEnabled`: Specifica lo stato di regolazione dell'attrezzatura.  True è regolante, false è non regolante. Predefinito: False  - `dataProvider`: Una sequenza di caratteri che identifica il fornitore dell'entità di dati armonizzata.  - `dateCreated`: Timestamp di creazione dell'entità. Questo sarà di solito assegnato dalla piattaforma di archiviazione.  - `dateModified`: Timestamp dell'ultima modifica dell'entità. Questo sarà di solito assegnato dalla piattaforma di archiviazione.  - `description`: Una descrizione di questo articolo  - `highStep`: La più alta posizione possibile del passo di rubinetto, l'avanzamento dal neutro. L'attributo deve essere maggiore di lowStep. Predefinito: 0  - `id`: Identificatore unico dell'entità  - `location`: Riferimento Geojson all'elemento. Può essere Point, LineString, Polygon, MultiPoint, MultiLineString o MultiPolygon  - `lowStep`: Posizione di gradino più bassa possibile, ritardo dalla posizione neutra Default: 0  - `ltcFlag`: Specifica se un TapChanger ha o meno capacità di cambiare il carico. Predefinito: False  - `name`: Il nome di questo articolo.  - `neutralStep`: La posizione del passo del rubinetto neutro per questo avvolgimento. L'attributo deve essere uguale o maggiore di lowStep e uguale o minore di highStep. Predefinito: 0  - `neutralU`: Tensione alla quale l'avvolgimento funziona con l'impostazione del rubinetto neutro. Predefinito: 0.0  - `normalStep`: La posizione del passo di commutazione utilizzata nel funzionamento normale della rete per questo avvolgimento. Per un commutatore `fisso` indica l'attuale impostazione fisica del tap. L'attributo deve essere uguale o maggiore di lowStep e uguale o minore di highStep. Predefinito: 0  - `owner`: Una lista contenente una sequenza di caratteri codificata in JSON che si riferisce agli ID unici dei proprietari  - `seeAlso`: elenco di uri che puntano a risorse aggiuntive sull'elemento  - `source`: Una sequenza di caratteri che dà la fonte originale dei dati dell'entità come URL. Si raccomanda di essere il nome di dominio completamente qualificato del fornitore di origine, o l'URL dell'oggetto di origine.  - `step`: Posizione del commutatore. Passo iniziale per una soluzione a stato stazionario. I valori non interi sono permessi per supportare le variabili di rubinetto continue. Le ragioni per il valore continuo sono per supportare casi di studio in cui non è stato ancora progettato un commutatore discreto, soluzioni in cui una stretta banda di tensione costringe il passo di commutazione ad oscillare o per adattarsi ad una soluzione continua come input. L'attributo deve essere uguale o maggiore di lowStep e uguale o minore di highStep. Predefinito: 0.0  - `type`: Tipo di NGSI. Deve essere TapChanger
